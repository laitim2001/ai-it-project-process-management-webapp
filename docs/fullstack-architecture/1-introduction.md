# 1. 介紹 (Introduction)

本文件基於已批准的 `prd.md` 和 `front-end-spec.md`，旨在為「IT 專案流程管理平台」定義一個統一、穩健且可擴展的全端技術架構。它將作為後續所有 AI 驅動開發和人工開發的最高技術指南。

我們選擇了 **T3 Stack** 作為核心技術棧，以充分利用其端到端的 TypeScript 類型安全、現代化的開發體驗和高效的全端整合能力。
